## OFIMÁTICA Y PAQUETES INTEGRADOS

## 1. PAQUETES INTEGRADOS (SUITES OFIMÁTICAS)
Un paquete integrado es un conjunto de aplicaciones de software diseñadas para trabajar juntas, compartiendo una interfaz similar y permitiendo el intercambio fácil de datos entre ellas. [1, 2] 

* Características:
* Interfaz común: Menús, iconos y atajos de teclado similares en todas sus herramientas.
   * Integración de datos: Capacidad de insertar una tabla de la hoja de cálculo en un documento de texto (tecnología OLE o vinculación).
   * Multitarea: Posibilidad de trabajar con varios documentos y aplicaciones simultáneamente.
* Ventajas:
* Facilidad de aprendizaje: Al aprender a usar una herramienta, ya se conoce gran parte del funcionamiento de las demás.
   * Coste: Suele ser más económico adquirir la suite completa que cada programa por separado.
   * Estandarización: Facilita el intercambio de archivos dentro de una organización. [3] 

------------------------------
## 2. PRINCIPALES HERRAMIENTAS OFIMÁTICAS## A. Procesadores de Textos (Ej. Microsoft Word)
Herramientas para la creación, edición y formato de documentos escritos. [4] 

* Funciones clave: Estilos, tablas de contenido automáticas, combinación de correspondencia, revisión ortográfica y control de cambios para trabajo colaborativo.

## B. Hojas de Cálculo (Ej. Microsoft Excel)
Programas diseñados para manipular datos numéricos y alfanuméricos dispuestos en tablas (celdas). [5] 

* Funciones clave: Fórmulas matemáticas, funciones lógicas y estadísticas, creación de gráficos, tablas dinámicas y macros para automatizar tareas.

## C. Gestión de Bases de Datos (Ej. Microsoft Access) [6] 
Software para organizar y administrar grandes volúmenes de datos relacionados. [7] 

* Funciones clave: Creación de tablas, formularios para introducir datos, consultas (queries) para filtrar información e informes para impresión. [8] 

------------------------------
## 3. MICROSOFT OFFICE
Es la suite ofimática líder del mercado. Actualmente conviven dos modelos:

* Office Tradicional (Perpetuo): Pago único por una versión específica (ej. Office 2021).
* Microsoft 365 (Suscripción): Modelo basado en la nube (SaaS) que incluye actualizaciones constantes, almacenamiento en OneDrive y herramientas de colaboración en tiempo real como Teams. [9, 10, 11, 12, 13] 

## Otros paquetes relevantes:

* LibreOffice / OpenOffice: Alternativas de Software Libre y gratuitas, compatibles con los formatos de Microsoft.
* Google Workspace: Suite 100% web centrada en la colaboración online. [14, 15, 16, 17] 

------------------------------
Dato clave: El formato de archivo estándar de Microsoft Office es el OpenXML (extensiones .docx, .xlsx, .pptx), que es un formato comprimido basado en XML para mejorar la interoperabilidad y reducir el tamaño de los archivos. [18, 19] 

Aquí tienes un resumen de los formatos de archivo más utilizados en ofimática. Es fundamental conocerlos para garantizar la interoperabilidad (que otra persona pueda abrir tu archivo) y la preservación a largo plazo.
## Tabla de Formatos de Archivo Ofimáticos

| Tipo de Documento | Formato Estándar (Cerrado/Propietario) | Formato Abierto (ENI / Libre) | Formato de Intercambio / Plano |
|---|---|---|---|
| Texto | .docx (MS Word) | .odt (OpenDocument) | .pdf (No editable / Difusión) |
| Hoja de Cálculo | .xlsx (MS Excel) | .ods (OpenDocument) | .csv (Solo datos, separado por comas) |
| Presentaciones | .pptx (MS PowerPoint) | .odp (OpenDocument) | .pdf (Para lectura) |
| Base de Datos | .accdb (MS Access) | .odb (LibreOffice Base) | .sql (Scripts de estructura y datos) |
| Texto Simple | - | - | .txt (Sin formato) / .rtf (Formato básico) |

------------------------------
## Conceptos clave para tu documento:

   1. PDF (Portable Document Format): Es el estándar para el intercambio de documentos que no deben ser modificados. Según el ENI, el formato PDF/A es el recomendado para el archivo y preservación a largo plazo de documentos administrativos.
   2. CSV (Comma Separated Values): Es el formato más simple para mover datos entre diferentes hojas de cálculo o bases de datos, ya que no guarda formatos ni fórmulas, solo los valores puros.
   3. ODF (Open Document Format): Es la familia de formatos abiertos (.odt, .ods, .odp) que las Administraciones Públicas deben ser capaces de leer y guardar para cumplir con el principio de neutralidad tecnológica.
   4. Macros (.docm, .xlsm): Son archivos que contienen scripts (automatizaciones). Ojo: suelen ser una vía de entrada de malware, por lo que muchos sistemas de seguridad los bloquean por defecto.

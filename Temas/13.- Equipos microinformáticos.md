# ARQUITECTURA DE EQUIPOS MICROINFORMÁTICOS

## 1. ARQUITECTURA BÁSICA (Modelo von Neumann)

Casi todos los equipos actuales siguen este esquema: CPU (procesamiento), Memoria RAM (almacenamiento temporal), Unidad de Entrada/Salida (periféricos) y el Bus (las "carreteras" por donde viajan los datos).

## 2. PROCESADORES (CPU)

Es el "cerebro" del sistema. Sus características clave son:

* Núcleos (Cores): Permiten realizar varias tareas simultáneamente.
* Frecuencia (GHz): La velocidad de procesamiento (ciclos por segundo).
* Memoria Caché: Memoria ultra rápida interna (L1, L2, L3) para datos de uso inmediato.
* Arquitectura: Actualmente predominan los 64 bits (x86-64).
* Fabricantes: Principalmente Intel (Core i3/i5/i7/i9) y AMD (Ryzen).

## 3. MEMORIAS

* RAM (Random Access Memory): Volátil (se borra al apagar). Es donde se cargan el SO y los programas abiertos. El estándar actual es DDR4 y el emergente DDR5.
* ROM (Read Only Memory): No volátil. Contiene el firmware básico como la BIOS o UEFI, que inicia el hardware.

## 4. DISPOSITIVOS DE ALMACENAMIENTO

* HDD (Hard Disk Drive): Discos mecánicos. Lentos pero con gran capacidad y bajo coste.
* SSD (Solid State Drive): Basados en memoria Flash. Mucho más rápidos y resistentes.
* SATA: Formato tradicional de 2.5".
* M.2 NVMe: Se conectan directamente a la placa base y ofrecen velocidades hasta 20 veces superiores al SATA.

## 5. TARJETAS DE EXPANSIÓN

* Tarjeta Gráfica (GPU): Se encarga del procesamiento de imágenes y vídeo. Posee su propia memoria (VRAM). Se conecta vía puerto PCI-Express.
* Tarjeta de Red (NIC): Permite la conexión a la red mediante cable (Ethernet RJ-45) o Wi-Fi.

## 6. INSTALACIÓN Y CONFIGURACIÓN

* Placa Base (Motherboard): Es el soporte físico donde se conectan todos los componentes. Incluye el Chipset, que gestiona la comunicación entre ellos.
* BIOS / UEFI: Es el software de bajo nivel que se ejecuta al encender el PC. Permite configurar el orden de arranque, la fecha/hora y parámetros de seguridad.
* Fuente de Alimentación: Convierte la corriente alterna en continua y suministra energía a cada componente según su voltaje (12V, 5V, 3.3V).

------------------------------
Dato clave: Para que un ordenador sea equilibrado, debe evitarse el "cuello de botella" (por ejemplo, tener un procesador muy potente pero un disco duro HDD lento, lo que ralentizaría todo el sistema).
